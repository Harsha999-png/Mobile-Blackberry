# Mobile BlackBerry Emulator

An interactive web-based recreation of the classic BlackBerry Curve 2 phone. Built with pure HTML, CSS, and JavaScript, this project recreates the look and feel of the iconic BlackBerry device — complete with a working on-screen QWERTY keyboard, navigable menu, and live status bar.

## Live Demo

Visit the live site: https://HarshaSreeG.github.io/Mobile-Blackberry/

## Features

- Realistic BlackBerry Curve 2 phone mockup with rounded frame and authentic styling
- Working on-screen QWERTY keyboard that captures typed text
- Trackpad navigation that cycles through 4 menu options: Messages, Contacts, Browser, Media
- Live-updating clock in the status bar
- Functional Shift, Alt, Space, Delete, and Enter keys
- Hardware-style call, menu, back, and end buttons
- Homescreen background image
- Pop-up alert displaying selected menu item and typed text on Enter

## Tech Stack

- HTML5
- CSS3 (no frameworks)
- JavaScript (Vanilla, no libraries)

## How to Run Locally

1. Clone or download this repository
2. Open the project folder
3. Double-click index.html to open it in your browser

No build step or server required.

## How It Works

- Keyboard buttons add characters to the input string and display them on screen
- Trackpad cycles through the 4 menu items, highlighting the active one
- Shift / Alt keys toggle between letter and symbol modes
- Delete removes the last typed character
- Enter triggers an alert showing the selected menu item and the text typed
- Clock in the status bar updates automatically every minute
